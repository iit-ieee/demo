To add or update events on this page, simply edit the js/events.js file. Add your event to the events array with the following information:

  • title: Name of the event
  • date: Date in YYYY-MM-DD format (e.g., "2025-12-15")
  • time: Time of the event
  • location: Venue or location
  • description: Brief description of the event
  • type: Category (Workshop, Hackathon, Tech Talk, etc.)
  • registerLink: Registration URL (optional)

The website will automatically sort events into "Upcoming" and "Past" based on today's date!
